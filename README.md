# StreamCast - Remote OBS Control via Discord

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Python](https://img.shields.io/badge/python-3.7+-blue.svg)](https://www.python.org/downloads/)
[![Discord](https://img.shields.io/discord/your-server-id?label=Discord)](https://discord.gg/your-invite-link)

StreamCast is a powerful tool that bridges Discord and OBS, allowing you to delegate streaming control to your moderators through Discord commands.

## Features

- **Scene Switching**: Change OBS scenes remotely via Discord commands
- **Stream Control**: Start/stop your stream with simple commands
- **Secure Access**: Role-based access control for moderators
- **Easy Setup**: Simple configuration through an intuitive interface
- **Auto-complete**: Scene suggestions for quick switching
- **Secure**: Local encryption for sensitive data

## Installation

For installation instructions, please visit our official website:
[StreamCast Installation Guide](https://stream-cast.netlify.app)

## Usage

StreamCast runs entirely in the cloud. Once you've set up your account on our website, you can control your OBS scenes and stream through Discord commands.

### Available Commands

- **Switch Scenes**: Change to any configured OBS scene
- **Stream Control**: Start or stop your stream
- **Moderator Management**: Manage who can control your stream
- **Scene Suggestions**: Get auto-complete for your scene names

## Security

- End-to-end encryption for all communications
- Role-based access control for stream management
- Secure OAuth2 integration with Discord
- Your OBS WebSocket password is never stored on our servers
